Pre-requisited
1. Download and install latest Node js
2. Download vscode

Creating Angular Project
1. install angular: npm install -g @angular/cli
2. fake json server: npm install -g json-server
3. go the folder you want to create project, then open cmd from there
4. cmd to create project : ng new project-name
5. Open project from vs code
6. bootstrap setup

Steps to the local server
1. Open terminal from your project
2. Run npm i -g json-server - it download the required package for running local server
	and create a json file db.json which store the ARRAY OF Object 
3. Run server: json-server --watch db.json
